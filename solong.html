So long

Suramya Pokharel - December 15, 2023

My eyes have called so long
So long for another sight
So long for the golden days
So long for the mystical night
So long have these words flown
So long has the fire blown
So long have I hurled myself
Hurled myself with the gloomy fist
and yet I find me standing here
Trying to look beyond the mist.

These thoughts have wandered so long
So long in the african sands
So long in the doors of hell
So long in the heavenly lands
So long have I seeked the views
So long have I felt the news
So long have I woken myself
Awake form a deadly dream
and yet I find myself dreaming again
Seeking the view beyond the beam

Darkened have the days so long
So long has the sun been dim
So long has the horizon hid
So long have I been this grim
So long have I called the rays
So long have I sat and gazed
So long have I peeked the mind
And looked the truth in the eyes
And yet I find myself believing again
Loving all these excellent lies
