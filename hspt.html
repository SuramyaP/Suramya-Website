Hospital Management System

Abstract:

Hospital Management System focuses on the management of data inside a hospital. Using SQL as the primary backbone of the project, there was a special focus on storing data of doctors, patients and assigning patients to respective doctors based on time shift and then informing patients with their appointment turn. HTML, CSS and Django were used in conjunction to create a responsive website that had three potential end users in Patients, Doctors and Admin. Python being the primary programming language used, the end product was an easy to use effective website with a swift functioning backend.
